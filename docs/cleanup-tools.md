# 页面维护工具

cleanup 是执行器里的页面小工具，不依赖任务包，也不会参与 AI 分类或视频迁移计划。

## 清理失效视频

入口：`更多` -> `清理失效视频`

行为：

- 读取“我创建的收藏夹”。
- 弹窗列出可识别收藏夹 ID 的自建收藏夹。
- 默认全选。
- 对选中的收藏夹逐个调用 Bilibili clean 接口。
- 不分页扫描视频，不逐条判断失效视频。
- 每个收藏夹之间有随机间隔。
- 遇到 403/412/login/csrf/写接口异常立即停止。

适合用于清理 Bilibili 已判定的失效内容。

## 删除收藏夹

入口：`更多` -> `删除收藏夹`

行为：

- 读取“我创建的收藏夹”和视频数量。
- 默认全不选。
- 空收藏夹和非空收藏夹都可以手动勾选。
- 提供搜索、只看已选、全选可见、可见全不选、反选可见。
- 删除前必须输入确认短语。
- 删除会连同夹内收藏关系一起移除。
- 删除成功后页面会刷新，重新读取侧栏。
- 遇到 403/412/login/csrf/写接口异常立即停止。

## 边界

- cleanup 不读取任务包。
- cleanup 不保护任务包目标夹。
- cleanup 不做视频分类。
- cleanup 不写知识库。

删除收藏夹前请确认名称和视频数量，并导出备份。
